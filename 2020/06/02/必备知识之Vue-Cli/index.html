更快速构建vue项目

快速原型开发

你可以使用vue serve和vue build命令对单个vue文件快速进行原型开发

安装@vue/cli-service-global扩展

npm install -g @vue/cli-service-global

准备一个内容原型

vue serve

启动一个服务并运行原型

vue serve Hello.vue

创建项目

vue create

创建一个vue项目

vue create projectname

vue ui

图形化项目管理

vue ui
