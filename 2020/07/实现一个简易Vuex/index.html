在阅读本篇之前，希望你已经对vuex的用法有了一定了解。可以参考一下Vue统一状态管理——Vuex。

Vuex

Vuex集中式存储管理应用的所有组件的状态，并以相应的规则保证状态以可预测的方式发生改变

整合vuex

核心概念

  • state 状态、数据
  • mutations 更改状态的函数
  • actions 异步操作
  • store 包含以上概念的容器

状态 - state

state保存应用状态

状态变更 - mutations

mutations用于修改状态

派生状态 - getters

从state派生出新状态，类似计算属性

动作 - actions

添加业务逻辑，类似于controller
